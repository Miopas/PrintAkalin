# PrintAkarin
在命令行打印一个阿卡林。

用 python 的图片处理库来打印线稿类型的图片。

线稿可以用这个工具提取：線稿提取器[https://github.com/FlandreDaisuki/Outline-Extractor]

# Requirements
* Python3
* numpy
* pillow

# Example
`python run.py images/akarin_large.png`

![img](https://github.com/Miopas/PrintAkalin/raw/master/example.gif)
